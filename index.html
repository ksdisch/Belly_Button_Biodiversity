The bar chart (above) displays the top 10 bacterial species (OTUs) found in the belly button of the selected subject. The x values represent the amount of each bacteria found, the y values display the id OTU identification number, and if you hover over a bar, the species name(s) are displayed.



The gauge chart (above) represents the frequency with which the selected subject reported washing their belly button between 0 times and 10+ times per week.


The bubble chart displays the different bacterial species, as well as the amount of each species found in the belly button of the selected subject. The x values represent the different OTU identification numbers, the y values represent the amount of each bacteria found, the size of each bubble is proportional to the amount of bacteria found, and if you hover over a bubble, the species name(s) are displayed.
